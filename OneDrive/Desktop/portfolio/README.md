# Spandan Behera - Portfolio

Welcome to my personal portfolio repository! 

🚀 **Live Link:** [https://portfolio-woad-two-65.vercel.app](https://portfolio-woad-two-65.vercel.app)

## Tech Stack
- Frontend: React (Vite) + Tailwind CSS
- 3D Animations: React Three Fiber + Drei
- Animations: Framer Motion
- Deployment: Vercel

## Running Locally

To run this project on your own machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/Spandeeee/-portfolio.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
